# Telco Customer Churn Analysis

## Overview
An exploratory data analysis and interactive Power BI dashboard 
built on the Telco Customer Churn dataset. This project analyses 
customer behaviour to understand why customers leave and what 
keeps them engaged.

## Tools Used
- Python (Pandas, Matplotlib) — data cleaning and EDA
- Power BI — interactive churn dashboard
- GitHub — version control and documentation

## Files
| File | Description |
|---|---|
| `TELCO CHURN DASHBOARD - TTK CHIKWANE.pbix` | Power BI dashboard file |
| `telco_churn_clean.csv` | Cleaned dataset |
| `TTK Chikwane - Task2.ipynb` | Python EDA notebook |

## Key Insights
- Overall churn rate is 26.54%
- Month-to-month customers churn the most
- Fiber optic internet users have the highest churn
- Electronic check payment method has the highest churn rate
- Customers with higher monthly charges are more likely to churn
- Churn rate decreases significantly as tenure increases

## Recommendations
- Incentivise month-to-month customers to switch to annual contracts
- Investigate why Fiber optic customers are leaving
- Offer loyalty rewards to customers in their first 12 months
- Review pricing for high monthly charge customers
- Promote auto-payment methods to reduce churn risk
